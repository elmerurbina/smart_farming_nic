## 📌 Descripción del Proyecto

**Smart Farming Nicaragua** es un sistema IoT de riego automatizado diseñado para optimizar el uso del agua y monitorear las condiciones del suelo en cultivos agrícolas.

El sistema utiliza sensores de humedad del suelo y sensores NPK para analizar las condiciones del terreno en tiempo real, y activa automáticamente electroválvulas para el riego inteligente.

El proyecto está diseñado con una arquitectura modular y escalable, inspirada en principios de desarrollo backend tipo Django, permitiendo su expansión hacia sistemas web y cloud.

---

## ⚙️ Características

- 🌡️ Monitoreo de humedad del suelo en tiempo real  
- 🧪 Lectura de nutrientes NPK (N, P, K)  
- 🚰 Control automático de electroválvulas  
- 📊 Visualización en Serial Monitor  
- 📈 Soporte para Serial Plotter  
- 🖥️ Pantalla OLED SSD1306 (I2C)  
- 📡 Preparado para IoT (WiFi / MQTT / API)  
- 🧩 Arquitectura modular escalable
